相比于 Android 用户拥有的众多免费或开源代理工具，iOS 用户的选择更为受限。而 Shadowrocket，凭借其在 iOS 平台上的稳定性、灵活性、功能全面性，成为了多数资深“翻墙”用户的首选。

  • 可与多种订阅地址自动同步节点

二、Shadowrocket 安装指南

1. 安装前提与设备要求

要顺利安装 Shadowrocket，需满足以下条件：

  • 一部运行 iOS 12 或更高版本的 iPhone/iPad

  • 绑定国外 Apple ID（Shadowrocket 仅上架在部分国家/地区的 App Store，例如美国、香港）

  • 足够的 Apple 账户余额（购买需一次性支付约 2.99 美元）

2. 获取 Shadowrocket 的正确方式

由于该软件在中国区 App Store 下架，用户需通过以下方式安装：

  • 注册或借用一个非中国区 Apple ID（如美区或港区）

  • 登录 App Store，搜索“Shadowrocket”

  • 点击购买并安装，下载完成后即可开始使用

⚠️提示：切勿使用第三方破解版本或来源不明的安装包，可能存在信息泄露风险。

三、首次运行 Shadowrocket 的准备工作

首次打开应用时，Shadowrocket 会请求以下权限：

  • VPN 设置权限（需允许系统配置 VPN）

建议在“设置”中打开后台运行、允许使用蜂窝数据等功能，以确保其持续工作。

四、Shadowrocket 核心配置详解

1. 添加代理配置（手动或订阅）

方法一：手动添加节点

适用于已有代理信息的用户：

  • 打开 Shadowrocket，点击右上角 “+” 号

  • 选择代理类型（如 Shadowsocks、VMess 等）

  • 输入服务器地址、端口、密码、UUID 等参数

  • 命名该节点并保存

方法二：通过订阅链接导入

适用于使用商用代理服务的用户：

  • 获取代理商提供的订阅链接（通常以 http 或 https 开头）

  • 进入 Shadowrocket > 配置 > 订阅 > 添加订阅

  • 粘贴链接，点击“完成”即可自动同步所有节点

订阅更新可以定时自动进行，也可以手动刷新。

2. 设置规则分流（推荐使用）

规则决定哪些网站走代理，哪些直连，是优化上网体验的关键。

常见规则模式：

  • 全局模式：所有流量均通过代理；

  • 代理名单模式：仅指定网站通过代理（如 Google、YouTube）；

  • 绕过中国模式：除境内网站外，其余全部代理（推荐）；

  • 自定义规则：按需设定特定域名/IP 规则。

Shadowrocket 支持导入外部规则文件，例如 GitHub 上的 Loyalsoldier 规则集合，也支持用户手动编辑。

FAQ

Sing-box 如何实现精确域名分流？
Sing-box 可以按域名设置分流规则，将不同流量发送到指定节点。结合策略组使用，可实现网页、视频和游戏流量精准分配，提高访问速度和网络稳定性。
QuantumultX 如何实现按端口和协议分流？
QuantumultX 支持按 TCP、UDP、端口及应用协议分流流量，结合策略组选择最优节点。可为游戏、视频和广告屏蔽提供最佳网络体验，提高访问速度和稳定性。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
VLESS 与 VMess 的主要区别是什么？
VLESS 是 VMess 的轻量化替代协议，移除了内置加密逻辑，更依赖 TLS 层进行安全保护。这种设计降低了服务器压力，提升了传输效率，并增强了可扩展性。
Hysteria2 与传统代理协议有何不同？
Hysteria2 使用基于 QUIC 的高性能传输协议，可显著降低延迟并提升抗干扰能力。它特别适合高丢包或高延迟网络环境，提供流畅的浏览和视频播放体验。
QuantumultX 支持哪些定制化分流策略？
QuantumultX 支持域名、IP、端口、协议以及用户自定义脚本分流。结合策略组使用，可实现广告屏蔽、视频加速、游戏优化等功能，提高访问速度和个性化网络体验。
Clash 如何防止规则文件冲突导致分流异常？
合理组织规则顺序，将更具体的规则放在前面，通配规则放在末尾；使用 rule-providers 分离不同来源的规则，避免重复条目，并在变更后使用日志或 debug 模式检验分流是否按预期生效。
Nekoray 支持节点订阅批量更新吗？
支持。用户可通过订阅 URL 批量更新节点，程序会自动下载并替换旧节点。定期更新保证节点列表始终可用，减少手动维护，提高使用效率和便利性。
Nekoray 如何实现节点延迟自动切换？
Nekoray 支持配置策略组（Policy Group），用户可设置 URL 测试或周期检测机制，根据延迟自动选择最优节点。此功能能提升稳定性，尤其适合多线路机场订阅用户。
Nekoray 支持 Trojan 吗？
支持。Nekoray 基于 sing-box 核心，能兼容 Trojan、Vmess、Vless、Hysteria2 等多种协议。用户可在同一界面中管理和切换不同类型节点，非常方便。
