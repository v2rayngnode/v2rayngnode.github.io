typescript环境搭建及编译运行

  • 首页
  • 新闻资讯
  • 正文

1.安装node，安装成功后  node -v查看node版本 npm -v  查看npm版本。

2.安装typescript包  npm install typescript -g       查看ts版本   tsc --version

3.新建一个文件夹，用于存放tsc代码。

    通过ctrl+r输入cmd打开dos命令行，打开存放tsc文件的目录，npm init -y生成package.json。

    通过  tsc --init 生成tsconfig.json，用于生产tyscript的项目配置文件。

4.安装types/node，npm install @types/node --dev-save  这个主要是解决模块的声明文件问题。

5.编写HelloWorld.ts文件

var a:string = "Helloworld"; console.log(a);

6.在vscode中 可以使用打开运行生成任务，然后选择tsc：构建-tsconfig.json，这时候就会生成一个helloWorld.js文件（如果使用sublime  可以在dos命令行中输入 tcs  HelloWorld构建helloworld.js文件）。

7.运行node  helloworld.js 输出“HelloWorld”，至此，typescript运行环境搭建完成
